# n8n Automation Templates

This comprehensive collection of resources is designed to help you navigate the world of AI automation and maximize efficiency in your business. As part of Amplify Automation's commitment to sharing knowledge, all content here is freely available to help businesses and individuals harness the power of automation.

Whether you're new to automation or looking to enhance your existing processes, you'll find valuable guides, technical documentation, and best practices organized in the sections below.

## Templates

### Chatbots

AI-powered chatbot workflows for customer engagement and support.

| Template | Description |
|----------|-------------|
| `basic-website-chatbot.json` | A website chatbot with FAQ handling and appointment booking integration. Includes Google Calendar integration for scheduling. |
| `ai-chatbot-rag-crm-live-sync.json` | Advanced AI Agent chatbot with RAG (Retrieval-Augmented Generation) and CRM synchronization capabilities. |

### Receptionist

AI virtual receptionist workflows for handling inbound calls and inquiries.

| Template | Description |
|----------|-------------|
| `electrical-inbound-receptionist.json` | Retell AI-powered inbound call flow with Cal.com integration for appointment management. |

### Speed to Lead

Automated lead capture and instant response workflows to improve conversion rates.

| Template | Description |
|----------|-------------|
| `website-contact-form-speed-to-lead.json` | Captures contact form submissions and triggers immediate follow-up actions. |
| `n8n-contact-form-to-instant-response.json` | Built-in n8n form with lead capture and instant automated response system. |

## Getting Started

1. Download the JSON template file you need
2. Import it into your n8n instance via **Workflows > Import from File**
3. Configure the required credentials (OpenAI, Google Calendar, Cal.com, etc.)
4. Customize the workflow to match your business requirements
5. Activate and test your workflow

## Requirements

- [n8n](https://n8n.io/) (self-hosted or cloud)
- API credentials for integrated services (varies by template)
